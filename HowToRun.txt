Use this clean setup checklist on a new laptop (Windows-focused, matching your current project setup).

**1. Install Prerequisites**
1. Install Git.
2. Install Docker Desktop and make sure it starts.
3. Install Python 3.12 (or 3.11+).
4. Install Node.js 20 LTS (includes npm).

**2. Clone Project**
1. Open PowerShell.
2. Run:
   git clone <your-repo-url>
   cd AGC

**3. Create Python Virtual Environment**
1. From project root:
   python -m venv .venv
2. Activate:
   Activate.ps1
3. Install backend deps:
   cd backend
   ..\ python.exe -m pip install --upgrade pip
   ..\ python.exe -m pip install -r requirements.txt

If PowerShell blocks activation, run once as admin:
Set-ExecutionPolicy RemoteSigned

**4. Install Frontend Dependencies**
1. In another terminal:
   cd AGC\frontend
   npm install

**5. Configure Environment**
1. Confirm values in .env.
2. Keep extractor in scaffolded mode initially:
   DOCUMENT_EXTRACTOR_PROVIDER=scaffolded
3. You can configure external OCR/NLP later.

**6. Start Infrastructure (Docker)**
1. From AGC root:
   docker compose up -d postgres neo4j redis
2. Confirm healthy:
   docker compose ps

**7. Run Database Migrations**
1. In backend folder:
   c:/path/to/AGC/.venv/Scripts/python.exe -m alembic upgrade head

**8. Start App Services**
1. Backend API:
   c:/path/to/AGC/.venv/Scripts/python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --reload
2. Frontend:
   npm run dev
3. Celery worker (required for background document jobs):
   c:/path/to/AGC/.venv/Scripts/python.exe -m celery -A app.core.celery_app:celery_app worker --loglevel=info --pool=solo

**9. Verify It Works**
1. Backend health:
   http://127.0.0.1:8000/health
2. Frontend:
   http://localhost:3000
3. Optional test run:
   cd AGC\backend
   c:/path/to/AGC/.venv/Scripts/python.exe -m pytest -q

**Project Runbook Files**
- Startup guide: commands.txt
- Env config: .env
- Container definitions: docker-compose.yml
- Current implementation plan: execution-plan.md
